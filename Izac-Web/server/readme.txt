db.json API

View all Passangers : GET : http://localhost:9000/passangers
View a Passanger : GET : http://localhost:9000/passangers/:passangerId
Create Passanger : POST : http://localhost:9000/passangers/
Update a Contact : PUT : http://localhost:9000/passangers/:passangerId
Delete a Contact : DELETE : http://localhost:9000/passangers/:contactId
